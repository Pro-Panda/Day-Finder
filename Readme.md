## Day-Finder
Algorithm to find the day of any date

### Usage:

To compile
```shell
gcc -Ofast src/day_finder.c -o day_finder
```

To run the script
```
./day_finder
```

### Demo:
```shell
Enter date [dd-mm-yyyy]:
18-07-2018
It is Wednesday
Retry [y/n] ?
y
Enter date [dd-mm-yyyy]:
18-07-2018
It is Wednesday
Retry [y/n] ?
n
```
